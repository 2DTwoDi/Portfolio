# Filmage Media — Portfolio + Tools

Structure:
/ (root)

- index.html → Main portfolio (Work, Clients, About, Services, Contact)
- tools.html → Filmage Tools page (free editor tools)
- css/
  - style.css → Main styles (dark cinematic theme)
  - tools.css → Tools page extra styles
- js/
  - main.js → Portfolio logic (filters, modals, cursor, toast)
  - tools.js → Tools page logic (compress, extract, frames, etc)
- assets/ → Put logos, thumbnails, videos here

Navbar:

- index.html has Tools link → tools.html
- tools.html has back links to index.html sections

Clients section: Removed "Trusted by..." heading, now just "Selected Clients"

To run: Just open index.html in browser, or host folder on Netlify/Vercel.
